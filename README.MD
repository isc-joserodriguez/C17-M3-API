Pasos para correr el proyecto:
1. Instalar dependencias 

``` bash
npm install en la raíz
```

```bash

PORT = #Puerto de la API
MONGO_URI = #Conexión a mongo
JWT_SECRET = #Secret para los tokens
```


3. Correr proyecto:

``` bash
npm run dev
```
